# Ogun State electoral geography (LGAs → Wards → Polling Units)

Reference dataset for the **State → LGA → Ward → Polling Unit** hierarchy used by
[CR-0002](../../change-requests/0002-polling-units-registration-and-live-count.md).

## Files

| File | Description |
|------|-------------|
| `ogun_polling_units.csv` | Flat rows: `state, lga_code, lga_name, ward_code, ward_name, pu_code, pu_name` |
| `ogun_polling_units.json` | Nested `LGA → ward → polling_units[]` (each with `code`, `name`) |

## Source (authoritative)

**INEC — _Directory of Polling Units_, Ogun State, Revised January 2015.**
Downloaded from the Independent National Electoral Commission:
`https://www.inecnigeria.org/wp-content/uploads/2019/02/PU_Directory_Revised_January_2015_Ogun.pdf`

This is INEC's own delimitation publication. The data was **extracted programmatically** from the
PDF (via `pdftotext -layout`), not hand-transcribed or generated by a model.

## Totals

| | Count |
|---|---|
| LGAs | **20** |
| Wards (Registration Areas) | **236** |
| Polling units | **3,213** |

## Verification performed

The extraction was cross-checked against INEC's **own** figures inside the same PDF (checks that
did not use the row-parser's output):

- The 236 per-ward `TOTAL PUs:` values printed in the PDF **sum to 3,213**, and each matches the
  parsed per-ward row count **in document order** (all 236 match).
- All 20 LGAs match the PDF's state summary table on both **# of RAs** and **# of PUs**.
- Every ward's polling-unit codes run **contiguously 1..N** — no gaps or duplicates.
- Verbatim spot-checks of parsed PU names are present in the source text.
- The JSON and CSV are mutually consistent (20 / 236 / 3,213).

## Important caveat — 2015 delimitation, not post-2022

This reflects the **3,213** polling units as of the **January 2015** revision. In **2022** INEC
expanded polling units nationwide; Ogun grew from **3,213 → 5,030** PUs. INEC has **not** published
the ~1,817 new PUs as a single downloadable named directory — they are exposed only via the online
PU Locator / IREV portals. If the project needs the post-2022 set, that data must be sourced
separately and this file updated.

Note: LGAs **"Egbado North / Egbado South"** in the INEC codes are the areas since renamed
**"Yewa North / Yewa South"**.
